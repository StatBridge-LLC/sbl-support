Server Status

This section covers information displayed on the Oxygen ‘Server Status’ page.​

To access this page, click “Server Status” on the Dashboard. This will load the ‘Server Status’ page.

The following information is presented on this page:

Server​

This area describes the core Oxygen software that is currently installed on your system. This includes the core Oxygen functionality that exists on your system. The following two items are displayed:

• Version: This is the version of the server software that’s installed on your system.

• Build Date: This shows when the software version was published by StatBridge.

Customer​

This area describes the client-focused software that is currently installed on your system. This includes any specific features or customizations that have been programmed for a client. The following three items are displayed:

• Name: This is the customer name as StatBridge has recorded it.

• Version: This is the version of the customer-specific software that’s installed on your system.

• Build Date: This shows when the software version was published by StatBridge.

Note: The customer configuration software version is different than the core Oxygen software version. Your core Oxygen version can be updated without requiring any customer configuration updates, and vice versa.

Unit​

• Instance: This is the machine name given to your Oxygen mini-server.

• Unsent Billing: This refers to the number of participants that have yet to be counted since the last time your system was queried for billing numbers.

Network Interface MAC Address​

This identifies each physical connection your Oxygen system has for connecting to a local or Internet network. StatBridge will occasionally reference this when troubleshooting. There will typically be three addresses showing, though you won’t need to know these at any point during typical Oxygen usage.

Note: There are no management functions provided on this page. These details are presented for informational purposes, and to assist StatBridge when providing support for your Oxygen system.
