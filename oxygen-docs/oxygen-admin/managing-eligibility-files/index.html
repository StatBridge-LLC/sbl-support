Managing Eligibility Files

This section covers adding, editing and deleting eligibility files on the Oxygen system.​

To access this page, click on “Eligibility Files” from the Dashboard. This will load the ‘Manage Eligibility Files’ page.

Uploading An Eligibility File​

1. Connect your PC to the Oxygen system's wireless network. The network name will match the unique code name found on the top or side of your mini-server.

2. Launch the Chrome browser on your computer, and navigate to: https://oxygen.mystatbridge.com/

3. Log into the application, and click on ‘Eligibility Files’ in the Dashboard.

4. Click the “Add New” button.

5. Enter a reference name for the eligibility file.

6. Select the desired client. There will usually be two options: ‘Demo / Training’ and one specifically provided by StatBridge for your clients. Your selection here will determine which eligibility file(s) are visible when creating a new screening event.

7. Click “Choose File”, and browse and select the desired CSV file on your computer. NOTE: the best format for an eligibility file is “CSV UTF-8”. If you’re not sure, simply open your file in Microsoft Excel and choose ‘Save As’. From that menu, choose the “CSV UTF-8 (Comma delimited) (*csv)” option and save the file.

8. With the desired file showing to the left of ‘Choose File’, click the “Save” button.

→ The file will be uploaded... When finished, “Eligibility file uploaded successfully” will appear in the right corner of the screen for a few seconds (and the progress bar will display as complete).

Note 1: If any errors are found in the file, they will be displayed below. This could include things like an incorrectly formatted birthdate, or perhaps a mislabeled gender. Oxygen will indicate the row in the CSV file where an error exists. It is recommended to fix any errors in the CSV file, and then upload the corrected version of your file as a new upload starting with Step 4 above.

Note 2: If a screening is active and data is saved for at least 1 participant, loading an eligibility file for that event will no longer be possible. Therefore, it’s recommended that an eligibility file is uploaded and attached to a screening event before it's first activated.

Editing An Eligibility File​

1. On the ‘Manage Eligibility Files’ page, click the “Edit” button.

2. You now will be able to change the ‘Name’ or the selection for ‘Client' if desired.

3. When finished, click the “Save” button.

→ Your changes to the file name or client selection will now be applied to the eligibility file.

Note: It is not possible to re-upload an eligibility file once it has been uploaded via an ‘Add New’ entry. If you don’t like an existing eligibility file, you must create a new eligibility file entry and upload the desired file as per the steps above.

Deleting An Eligibility File​

1. On the ‘Manage Eligibility Files’ page, click the “Delete” button in the row of the file you wish to remove.

2. On the ‘Confirm eligibility file deletion’ window that appears, click “Delete”.

→ The eligibility file will be deleted.

Note: When an eligibility file is attached to a screening and the screening is activated for the first time, the eligibility file is copied to that screening event. Once this has happened, deleting an eligibility file will only remove it from the ‘Manage Eligibility Files’ page. It will not remove the eligibility file from that screening event.
