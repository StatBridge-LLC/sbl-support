Managing The Current Screening

This section covers activating, viewing information about, and concluding a screening event.​

To access this page, click “Current Screening” on the Dashboard. This will load the ‘Current Screening’ page.

Activating a Screening Event for the First Time​

1. First, verify that no screening is currently active. At the top of the ‘Current Screening’ page, the text will be shown: "No currently active screening. Select a screening to activate."

2. If an eligibility file will be used, verify that the file is connected to this screening event. To do so, navigate to Dashboard > Screenings and click “Edit” for the desired screening. An attached eligibility file will be displayed in the ‘Eligibility File’ selector.

3. Now verify the basic information about this screening event. Navigate to the ‘Current Screening’ page, and view the available screenings. For the desired event, confirm that the name and screening dates match the desired values. NOTE: to appear on this page, a screening event’s date range must include today's date. If the current date is outside this range, it won't appear as an available event on the 'Current Screening' page.

4. To activate a screening, click the button corresponding to the desired screening.

→ Message(s) will appear: "Screening activated" and "Loaded eligibility file" (if an eligibility file was attached in Step 2). The screening will now be active.

Note: When first activated, all attached eligibility file participants are loaded. If the screening is activated again later, the eligibility file will already be attached and will not be re-loaded.

Viewing Information about the Current Screening​

1. With a screening event currently active, navigate to the ‘Current Screening’ page from the Dashboard.

→ Information about the current screening will be displayed, such as ‘Screening Name’, ‘Start Date’, ‘End Date’, and the ‘Eligibility File’ being used.

Concluding an Active Screening​

1. Navigate to the ‘Current Screening’ page.

2. Click the “Conclude Screening” button.

3. On the prompt that appears, confirm the action by clicking “Conclude”, or click “Cancel” to keep the current screening active.

→ If confirmed, the event is concluded and all users will be logged out of the application and sent to the ‘User Login’ screen.

Note: you will also see a message that the server has been restarted. This is simply an additional way to ensure that all information has been cleared from the system, and no artifacts from the concluded event will be seen when logging in again.
