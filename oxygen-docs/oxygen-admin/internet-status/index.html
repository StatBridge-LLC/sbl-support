Internet Status

This section covers managing connections to wireless Internet networks with the Oxygen system.​

To access this page, click “Internet” on the Dashboard. This will load the ‘Internet Status’ page.

Internet Overview​

Any time an Oxygen system has been connected to a wi-fi network, it will remember that network in the future. And whenever Oxygen is within range of a network it’s previously connected to, it will connect automatically when the system is turned on.

There are 3 main areas on this page:

• Internet Status. If the Oxygen system finds a known wi-fi network, it will attempt to connect and show that network after ‘Wifi Network Connected to’. And if the connection is successful, the ‘Connection status’ will show as “Connected”.

• Add/Update to unit address book. This allows you to add a new wi-fi network to the list of networks that Oxygen can connect to.

• Unit Address Book. This simply lists all the wi-fi networks that have been added on this system.

Note: It is important that you establish a reliable way to connect your Oxygen system(s) to a wi-fi network. This allows StatBridge to troubleshoot issues and push updates with the latest code, ensuring you have the best customer support and software versions available.

Connecting to a Wireless Internet Network​

1. First, make sure the Oxygen system is at a location that doesn't require navigating though a ‘splash screen’ to access the Internet (as commonly found in hotels or coffee shops). A typical home wi-fi network is ideal, and mobile wi-fi hotspots work as well. Office wi-fi networks are often fine, provided there isn’t a firewall set up to block certain wi-fi connections.

2. Turn on the Oxygen system, log in and navigate to the ‘Internet Status’ page.

3. Enter the name of the wi-fi network you'd like to join in the “Wireless SSID” field.

4. Enter the wi-fi network's password in the “Passphrase” field.

5. Click the “Add” button.

6. Now click ‘Dashboard’ in the left sidebar to return to the main Oxygen dashboard page. Once there, click the ‘Internet’ tile again.

→ The ‘Internet Status’ page will be loaded, and the new wi-fi network name will be displayed below ‘Unit Address Book’. If the connection is successful, the ‘Internet Status’ section will also reflect this.

Note: If a wi-fi network’s password changes, or was mistyped when adding it initially, you can create a new entry starting with Step 3 above. When turned on, Oxygen cycles through all address book entries until it finds a connection that works. As such, multiple entries with the same ‘Wireless SSID’ name are not an issue so long as one of the passwords is correct.
